# 📖 최한실 이야기할머니의 전용 사이트

> 이야기할머니 최한실님을 위한 따뜻한 전래동화 전자책 웹사이트

![GitHub](https://img.shields.io/badge/version-1.0.0-orange)
![License](https://img.shields.io/badge/license-MIT-blue)

## 🌟 소개

이 웹사이트는 이야기할머니로 활동하시는 최한실님께서 편안하게 전래동화를 읽고 들려주실 수 있도록 특별히 제작된 시니어 친화형 전자책입니다. 17주차 분량의 한국 전래동화가 큰 글씨와 넓은 줄간격으로 구성되어 있어, 누구나 쉽고 편하게 이야기를 즐기실 수 있습니다.

## ✨ 주요 기능

### 📚 17가지 전래동화
- 송아지와 바꾼 무
- 해와 달이 된 오누이
- 흥부와 놀부
- 토끼의 재판
- 그 외 13가지 이야기

### 👵 시니어 친화 디자인
- **초대형 글씨**: 2.4rem (모바일 1.5rem)
- **넓은 줄간격**: 문장별 독립적인 줄 배치
- **따뜻한 색상**: 노란색과 주황색 계열의 밝고 편안한 디자인
- **손글씨 폰트**: 친근한 'Gaegu(개구)' 폰트 사용

### 🎯 편리한 기능
- ✅ 한 페이지당 4-5문장으로 읽기 편한 분량
- ✅ **🔊 이야기 듣기 (TTS)**: 브라우저 내장 음성 합성 기능을 이용한 이야기 구연 기능
- ✅ **문장 강조**: 현재 읽고 있는 문장을 시각적으로 하이라이트 표시
- ✅ 부드러운 Fade-In/Fade-Out 페이지 전환

## 🚀 사용 방법

### 로컬에서 실행
1. 저장소 클론
```bash
git clone https://github.com/yourusername/story-grandmother.git
cd story-grandmother
```

2. 브라우저에서 `index.html` 열기
```bash
# 또는 Live Server 등의 도구 사용
```

### GitHub Pages로 배포
1. GitHub 저장소 Settings → Pages
2. Source를 `main` 브랜치로 설정
3. 배포 완료 후 제공되는 URL로 접속

## 📂 파일 구조

```
story-grandmother/
├── index.html          # 메인 랜딩 페이지
├── reader.html         # 이야기 읽기 화면
├── home.css           # 랜딩 페이지 스타일
├── styles.css         # 리더 페이지 스타일
├── app.js             # 페이지 전환 및 내비게이션 로직
├── stories.js         # 17가지 이야기 데이터
├── logo.jpg           # 로고 이미지
├── photo.jpg          # 메인 사진
└── README.md          # 프로젝트 문서
```

## 🛠️ 기술 스택

- **HTML5**: 시맨틱한 구조
- **CSS3**: Flexbox, 애니메이션, 반응형 디자인
- **Vanilla JavaScript**: 프레임워크 없는 순수 JS
- **Google Fonts**: Gaegu, Gothic A1
